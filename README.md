# Blowfish

## 使用方法

### 初始化

```c
BLOWFISH_UINT8 key[8] = {0x01, 0x31, 0xD9, 0x61, 0x9D, 0xC1, 0x37, 0x6E};
Blowfish_Cipher cipher = {0};
Blowfish_InitCipher(&cipher, key, sizeof(key));
```

### 加密
```c
BLOWFISH_UINT8 plainText[8] = {0x5C, 0xD5, 0x4C, 0xA8, 0x3D, 0xEF, 0x57, 0xDA};
BLOWFISH_UINT8 result[8] = {0};
Blowfish_Encrypt(&cipher, result, plainText);
// result => {0xB1, 0xB8, 0xCC, 0x0B, 0x25, 0x0F, 0x09, 0xA0}
```

### 解密
```c
BLOWFISH_UINT8 cipherText[8] = {0xB1, 0xB8, 0xCC, 0x0B, 0x25, 0x0F, 0x09, 0xA0};
BLOWFISH_UINT8 result[8] = {0};
Blowfish_Decrypt(&cipher, result, cipherText);
// result => {0x5C, 0xD5, 0x4C, 0xA8, 0x3D, 0xEF, 0x57, 0xDA}
```
